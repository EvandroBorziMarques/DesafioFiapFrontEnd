# 	Front-end Aluno Turma

Desafio FIAP

## Features

- Front-end CRUD que contém: :heavy_check_mark:
  - Aluno
    - Criar aluno
    - Listar alunos
    - Buscar aluno por ID
    - Atualizar aluno
    - Deletar aluno
  - Turma
    - Criar turma
    - Listar todas as turmas
    - Buscar turma por ID
    - Atualizar turma
    - Deletar turma
